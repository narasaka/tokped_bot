tokped bot
==========
a lil cute bot that browses and collects data from tokopedia.com for you (it doesnt shop for you bcos u should gRoW uP tbh)
- - - -

## Installation ##
    
    git clone https://github.com/narasaka/tokped_bot.git
    cd tokped_bot
    sudo mv chromedriver /usr/local/bin/chromedriver
    sudo mv geckodriver /usr/local/bin/geckodriver
    sudo -H pip3 install -r requirements.txt

### Running the bot ###
    
    python3 tokped.py [something u want to search]
    
    example: python3 tokped.py topi bundar
    
### If you're confused lol ###
    
    python3 tokped.py -h
